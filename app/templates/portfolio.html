todayiread.xyz

March 2016 View source code

Developed over two weekends, todayiread.xyz helps you keep track of your reading progress. Set goals. Track what you’ve read. Visualize your reading habits with sweet graphs.

I developed todayiread.xyz so I could keep track of the reading I was doing for school and visualize my reading habits. I used to use Goodreads to keep track of my reading, but I found that it didn’t match up with the kinds of reading I did in grad school: I wasn’t reading books, I was reading chapters and journal articles, and Goodreads didn’t let you record those things. todayiread.xyz lets you track any kind of reading you want. This is still very much a work in progress: it's more of a proof of concept than anything right now.

Features

  • CRUD interface for managing reading sessions and user settings
  • Secure log in, email verification, user authentication
  • RESTful API
  • Data visualization using d3.js

Balladmonger

November–December 2015 View source code

Balladmonger is a small web app built with Python and Flask which uses Markov chains to dynamically generate poems—or “ballads”—based on one or more training texts. The instance deployed on Heroku is trained on King Lear and about two dozen other texts derived from or influenced by King Lear, ranging from early modern ballads to Sense and Sensibility to twenty-first century fanfiction. I developed it for a graduate seminar exploring the relationships between Shakespeare and early modern ballads as an attempt to remediate the early modern ballad aesthetic into a twenty-first century web app.

Features

  • RESTful API
  • Responsive UI
